chào mừng các cậu đã đến với đề tài course chatbot
luồng sự kiện chia thành 3 phần:
+ chào hỏi linh tinh( chưa làm, nó rất dễ)
+ trả lời bớt xàm lol đi tôi không hiểu ( chưa làm nó cũng rất dễ)
+ trả lời câu hỏi tìm kiếm môn học theo tên, mã lớp, mã học phần ( có thể viết sai chính tả) -> đã làm và cần test

- khi clone về các cậu cần vào thư mục sql import dữ liệu
- chỉnh sửa tài khoản , mk sql trong code. -> tự tìm
- có 2 file code chính : hellochatbot và mylogic... -> các file không quan tâm
- test chương trình ở file hellochatbot
- kết quả đầu ra chỉ bao gồm lớp nào, ãm lớp nào, mã học phần nào.-> chưa truy vấn từ db ra vì nó cũng dễ
- không xàm lol nữa sau đây là cách cài đặt cho may nào sử dụng ubuntu :
+ cài python3 : sudo apt-get install python3
+ cài pip3 cho python3: sudo apt-get install pip3
+ cài chatterbot cho python3 : pip3 install chatterbot
+ cài MySQLdb : sudo apt install python3-mysqldb
